• Describe what the DOM is and an example of how you might interact with it.

The DOM stands for the ‘Document Object Model’. It is like the physical skeleton map (or family tree) of the Document. It allow us to use Javascript to identify and manipulate specific parts of the documents. It is amazing cause you can pick up specific words in a document much like a search and find button because everything in a word document knows their seating arrangements. An example of how we use DOM would be if we were picking up certain words using the document.querySelector to find a word or statement and then adding .innerHTML to change it another statement.

• Explain the difference between accessing data from arrays and objects.

Accessing data requires different method depending if its an object or an array of objects. Accessing the elements (object) of a document, we can use the getElementById() code. One way of getting an array of objects (technically it is a nodeList) would be the use of getElementsByClass().
